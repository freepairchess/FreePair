# Pairing Engine Folder

Place the BBP Pairings executable here during development:

- Windows: `bbpPairings.exe`
- macOS/Linux: `bbpPairings` (ensure it is executable: `chmod +x bbpPairings`)

Source: https://github.com/BieremaBoyzProgramming/bbpPairings
