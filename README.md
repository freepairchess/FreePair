# FreePair

A chess-tournament pairing app for tournament directors — opens SwissSys
`.sjson` files, drives FIDE Dutch pairings via the bundled
[bbpPairings](https://github.com/BieremaBoyzProgramming/bbpPairings)
engine, and exports USCF rating-submission DBF files.

## Download

**Windows installer** (x64, ~90 MB, includes the .NET 10 runtime and the
pairing engine — no other downloads required):

➡ **[FreePair-win-x64-Setup.exe](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Setup.exe)**

After download, double-click the file. Windows will warn *"unrecognized
app"* on first install — click **More info → Run anyway**. The app is
unsigned in this preview; signing comes in a later release.

No admin rights required (per-user install). Tested on Windows 10 / 11.

A portable zip (extract anywhere, no install) is also available on
the [releases page](https://github.com/freepairchess/FreePair/releases/latest).
