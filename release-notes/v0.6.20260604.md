# FreePair v0.6.20260604

FreePair is a cross-platform chess-tournament pairing application for Windows. It imports SwissSys 11 `.sjson` files, runs USCF Swiss and FIDE Dutch pairings (via FreePair's own USCF engine and the BBP Pairings FIDE engine, both bundled), and gives the TD end-to-end tools to pair, score, intervene, and publish reports.

This release is a small UI quality-of-life pass on top of the v0.5 USCF engine improvements.

## What's new in v0.6

- **Maximized window on launch.** FreePair now opens full-screen by default instead of the small ~800×450 Avalonia default, so the Pairings / Standings / Wall Chart grids have room to breathe right away. Standard title-bar controls still let you restore / resize.
- **Scrollable Settings tab.** The Settings tab now has a vertical scrollbar on the right when the form is taller than the window. Previously the bottom rows (Auto-update controls, Save button) could be clipped on smaller windows / lower resolutions with no way to reach them.
- **Transparent app icon.** The FreePair icon now has a transparent background instead of an opaque white square, so it looks clean on Windows 11's dark taskbar and in Alt-Tab.

## Carried over from recent releases

- **Settings → Auto-update Current version row + "Check now" button** so you can confirm what's installed and trigger an on-demand update check (added in v0.4 / v0.5).
- **USCF pairing engine fidelity improvements** — five engine fixes from a manual case-by-case review of the regression corpus, raising the harness from 529/584 to 538/584 passing tests with no regressions (v0.3.20260604).
- **`docs/USCF_DISCRETIONARY_DIVERGENCES.md`** tracks accepted SwissSys-vs-USCF divergences and deferred engine improvements so future work doesn't lose context.

## Installer

Self-contained Windows x64 installer (`FreePair-win-x64-Setup.exe`) and a no-install portable build (`FreePair-win-x64-Portable.zip`). Both bundle:

- the FreePair desktop application (Avalonia UI),
- the FreePair USCF pairing engine,
- the FIDE Dutch pairing engine (BBP Pairings),
- the .NET 10 runtime — **no separate runtime install required**.

The auto-updater (Velopack) is included. Open `Settings → Auto-update` to see your installed version and trigger an on-demand check.

## Known limitations

Unchanged from v0.5.20260604:

1. **Board-ordering preferences** — pairings, colours, and bye all match SwissSys; only the board numbering differs.
2. **SwissSys-discretionary choices that conflict with USCF rules** — SwissSys sometimes gives byes to unrated players when a rated alternative was available, or floats a non-lowest-rated player; FreePair correctly follows USCF rules. See `docs/USCF_DISCRETIONARY_DIVERGENCES.md` for the catalogue.
3. **Deferred engine improvements requiring larger refactors** — weighted conflict-cost model, absolute-board-number threading for cross-section colour tiebreaks, and floater-lock heuristic refinements. Also catalogued.
