# FreePair v0.42.20260718

A UI fine-tuning release focused on the **Event configuration** screen and the
**Pairings** tab, plus polish to the tiebreak and NA Chess Hub sync dialogs.

## What's new

### Event configuration
- **One tidy card, two columns.** The event's identity (title, dates, address,
  city/state/ZIP) and all of its settings now live in a single panel. Settings
  are laid out in two even columns so the screen is shorter and easier to scan.
- **Aligned, uncramped fields.** Labels are right-aligned next to their controls,
  label columns are widened so nothing gets cut off, the date range fills the row
  end-to-end, and the Organizer / NA Chess Hub boxes are roomier.
- **Simpler half-point-bye setup.** The per-section half-point-bye override table
  was removed from Event configuration — set each section's allowance in that
  section instead.

### Pairings tab
- **Pairing Operations menu.** Pair next round, Delete round, Open Pairing in
  Window, and Print Pairing are grouped into one dropdown to declutter the toolbar.
- **Options menu.** A "Sync Roster with NACH Before Pairing" checkbox (your choice
  is remembered) plus Advanced Options now live in a compact Options dropdown.
- **Consistent engine names.** The pairing engine picker uses the same
  "USCF (FreePair Engine)" / "FIDE (BBP Engine)" wording as Event configuration.

### Reordering & dialogs
- **Drag to reorder tiebreaks.** The Configure Tiebreaks dialog now supports
  drag-and-drop reordering (the Up/Down buttons still work).
- **Consistent drag handle.** The section list's reorder grip now matches the
  handle used elsewhere in the app.
- **Clearer NA Chess Hub sync review.** Columns size to their content so nothing
  is truncated. When a player is in FreePair but not on NA Chess Hub — usually an
  on-site registration — the dialog now shows a clear caution instead of an option
  that would remove them: **"Please manually delete this player from the roster if
  the player is not supposed to play. Otherwise he will be paired."**

## Installer

The Windows installer bundles the .NET runtime — no separate install required.
Download, run, and FreePair self-updates on future releases.

## Known limitations

- Half-point-bye allowances are now edited per section rather than in a single
  event-wide table (a deliberate simplification).
