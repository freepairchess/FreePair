# FreePair v0.39

This release is all about the **Roster tab** — giving you full control over
which columns you see, in what order, with support for every rating and ID a
NA Chess Hub file carries.

## What's new in v0.39

- **Choose your Roster columns — and their order.** A new **Optional Columns**
  button on the Roster tab opens a chooser where you tick the columns you want
  to see and drag them (or use ↑ ↓) into the order you like. Your arrangement
  is **saved with each section**, so every section can show exactly what its TD
  needs. Use **Show all** to reveal everything at once, or **Restore Default**
  to start over. The core columns (the action icons, ID, Name, Rating, and
  check-in) are always shown.

- **Every rating and ID, if you want them.** The chooser now exposes the full
  set of ratings and IDs that a NA Chess Hub file carries for each player:
  **USCF** (ID, regular, quick, blitz, and online regular/quick/blitz),
  **FIDE** (ID, standard, rapid, blitz), **CFC** (ID, regular, quick, blitz),
  and **Northwest** (ID, rating) — plus **Title**, **Sex**, and **Birth date**.
  Turn on just the ones you care about; they're all hidden by default so the
  grid stays clean.

- **Empty columns get out of your way.** A column you've turned on still hides
  itself automatically when no player in the section actually has a value for
  it — so enabling, say, the CFC columns for a US event won't leave a row of
  blank cells taking up space.

- **Tidier, better-fitting columns.** ID2, Rating2, and State now size to their
  contents (no more clipped values or a "State" header trimmed to "S"), the
  Team / half-point-bye / zero-point-bye columns shrink when empty, and the
  Club column is hidden by default (still searchable in the filter box).

- **FIDE country at a glance.** For FIDE-rated (and dual-rated) sections, a
  **FIDE Country** column shows each player's federation code.

- **Cleaner tables.** The Standings and Wall Chart grids no longer draw a
  doubled-up vertical line between the round columns.

## Installer

- **Download:** [FreePair-win-x64-Setup.exe](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Setup.exe)
- **Portable zip:** [FreePair-win-x64-Portable.zip](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Portable.zip)
- **Auto-update:** Existing FreePair installs (v0.3 and later) will pick up
  v0.39 automatically. If you want the update now, click "Check for updates"
  in the Settings tab.

The installer bundles the .NET 10 runtime, the BBP Pairings FIDE engine,
and the rules-reference document — no additional downloads or configuration
required.

## Known limitations (still planned for later releases)

- **Optional columns are display-only.** The extra rating and ID columns are
  shown and printed for reference; pairing still uses the section's pairing
  rating, exactly as before.
- **Column choice is per section.** Each section keeps its own optional-column
  arrangement; there is no single event-wide setting yet.
