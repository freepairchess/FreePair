# FreePair v0.45

A small **user-interface polish** release focused on the **Settings tab**. No
change to how any section pairs, scores, or reports — every event pairs exactly
as it did in the previous release. Upgrading is low-risk and recommended.

## What's new in v0.45

- **The Settings tab now looks like the rest of the app.** Each group of
  settings — pairing engines, display, online publishing, the online player
  database, tournament files, and auto-update — sits in its own bordered,
  lightly-shaded card, matching the Event configuration screen. A clear page
  title and one-line description sit at the top.
- **Settings save automatically.** The **Save** button is gone. Every setting
  now takes effect the moment you change it: checkboxes and the theme picker
  apply immediately, and text fields (folder paths, URLs) save as soon as you
  click or tab away. No more wondering whether a change "took."
- **More settings fit on screen.** The Settings content is wider now, so help
  text wraps far less and you can see most settings without scrolling.

## Bundled rules reference

The full USCF rules reference travels with this release, unchanged. Browse it at:

## Installer

- **Download:** [FreePair-win-x64-Setup.exe](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Setup.exe)
- **Portable zip:** [FreePair-win-x64-Portable.zip](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Portable.zip)
- **Auto-update:** Existing FreePair installs will pick up v0.45 automatically.
  To update now, click "Check for updates" in the Settings tab.

The installer bundles the .NET 10 runtime, the pairing engines, and the
rules-reference document — no additional downloads or configuration required.

## Known limitations

- **SwissSys fidelity is unchanged.** This is a UI-only release; the pairing
  engine and its regression coverage are exactly as in the previous version.
  Known divergences remain catalogued as future-work branches in
  `docs/USCF_RULES_COVERAGE.md`.
