# FreePair v0.34

This release is all about the look. Tournament directors told us the interface
felt **washed-out — "very pastel, nothing stands out."** FreePair now ships a
**Theme picker** so you can choose how it looks, and it remembers your choice on
that computer.

## What's new in v0.34

- **Pick a theme from the top of the window.** A new **Theme** dropdown lives at
  the top-right of the main window, right next to the Settings tab, so you can
  change the look from anywhere without digging through settings. Your pick takes
  effect immediately.

- **Themes that stand out.** Beyond the neutral **Light** and **Dark** options,
  there are four vivid accent themes — **Ocean Blue**, **Emerald**, **Sunset**,
  and **Grape** — plus a **High Contrast** theme for maximum legibility on a
  projector or a bright playing hall. The bold accent colour flows through the
  buttons, selected rows, checkboxes, and highlights so the important things on
  screen actually catch your eye.

- **Matches your Windows look by default.** The default **System** theme follows
  Windows automatically: it switches between light and dark with your Windows
  setting and even picks up your Windows accent colour. If you already run
  Windows in dark mode, FreePair keeps matching it — nothing changes unless you
  choose a theme yourself.

- **Your choice is remembered.** FreePair saves your selected theme on that
  computer and reapplies it every time you launch, so you set it once and forget
  it.

## Installer

- **Download:** [FreePair-win-x64-Setup.exe](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Setup.exe)
- **Portable zip:** [FreePair-win-x64-Portable.zip](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Portable.zip)
- **Auto-update:** Existing FreePair installs (v0.3 and later) pick up v0.34
  automatically. To update now, click "Check for updates" in the Settings tab.

The installer bundles the .NET 10 runtime, the BBP Pairings FIDE engine, and
the rules-reference document — no additional downloads or configuration
required.

## Known limitations (deliberate trade-offs, planned for later)

- **The vivid themes recolour the accent, not every panel.** Ocean Blue,
  Emerald, Sunset, and Grape change the accent colour that runs through buttons,
  selections, and highlights, and they still follow your light-or-dark
  background. The neutral information panels (blue notes, amber warnings, green
  success, red errors) keep their familiar colours on purpose so those signals
  stay recognisable at a glance.

- **The theme is a personal, per-computer preference.** Your choice is saved on
  the machine you set it on, not inside the tournament file — so moving an event
  to another computer won't carry the theme with it, and each TD can run their
  own preferred look.

- **"System" tracks Windows at launch.** The System theme picks up your current
  Windows light/dark setting and accent colour when FreePair starts. If you flip
  Windows between light and dark while FreePair is already open, pick the theme
  again (or relaunch) to resync.
