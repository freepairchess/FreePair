# FreePair v0.43.20260718

A small follow-up to v0.42 that tidies the **Pairings** tab toolbar for
round-robin events.

## What's new

- **Round-robin pairing actions live in the Pairing Operations menu.** For a
  round-robin (or quad) section, "Pair as Round-Robin" / "Pair as Quads" and,
  once paired, "Delete all rounds" are now inside the Pairing Operations dropdown
  alongside Open Pairing in Window and Print Pairing — no more loose toolbar
  buttons.
- **No delete-single-round for round-robin.** Round-robin events pair every round
  at once, so the individual "Delete round N" action is hidden for them (use
  "Delete all rounds" instead). Swiss events are unchanged — they still show
  "Pair Next Round" and "Delete round N".

## Installer

The Windows installer bundles the .NET runtime — download, run, and FreePair
self-updates on future releases.
