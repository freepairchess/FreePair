# FreePair v0.35

This release lets you **keep a backup copy of your event file in the cloud** and
**re-open it from another computer** — using your NA Chess Hub event. It's a
convenience for TDs who move between machines or want an off-site copy of the
working file, and it's kept clearly separate from publishing results for
spectators.

## What's new in v0.35

- **Save a copy of your event to the cloud when you close it.** When you close an
  event that's linked to NA Chess Hub, FreePair offers to upload a copy of your
  working tournament file so you can pick it up later on a different computer.
  You choose each time — **Save a copy to cloud then close**, **Skip and close**,
  or **Keep event open** — and you can tick **"Don't ask me again"** to stop the
  prompt (turn it back on anytime from the Settings tab). Your local file is
  always saved first, so you never lose anything if the upload can't reach the
  cloud.

- **Open a cloud-saved event from anywhere.** The **Open event** menu now has
  **Download and Open a Cloud-Saved Event**, with two ways in: **Browse
  cloud-saved events** to pick from a list of your events that already have a
  cloud copy, or enter the **event ID and passcode** directly. FreePair
  downloads the file to this computer and opens it so you can keep running the
  event.

- **Clearer New / Open menus.** The toolbar is now organised into two tidy menus
  — **➕ New event** (create manually, or create from an online registry) and
  **🔁 Open event** (open a local file, a recent event, or a cloud-saved event).
  The dialogs now say exactly what they'll do, so "creating a brand-new event"
  and "downloading a cloud-saved event" never look alike.

- **Backup, not publishing.** Saving a copy to the cloud stores your working
  tournament file so *you* can re-open it elsewhere. It does **not** post
  pairings or results for players and spectators — that's still the separate
  **Publish online** feature.

## Installer

- **Download:** [FreePair-win-x64-Setup.exe](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Setup.exe)
- **Portable zip:** [FreePair-win-x64-Portable.zip](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Portable.zip)
- **Auto-update:** Existing FreePair installs (v0.3 and later) pick up v0.35
  automatically. To update now, click "Check for updates" in the Settings tab.

The installer bundles the .NET 10 runtime, the BBP Pairings FIDE engine, and
the rules-reference document — no additional downloads or configuration
required.

## Known limitations (deliberate trade-offs, planned for later)

- **Cloud save happens when you close the event, not continuously.** Your local
  file is saved automatically on every change as always; the cloud copy is
  uploaded when you close the event (or when you choose to save one). It's a
  snapshot for portability and backup, not a live sync.

- **Requires an NA Chess Hub event and passcode.** The cloud save/open features
  work with events linked to NA Chess Hub. Events without those credentials keep
  working exactly as before, entirely from local files.

- **The close-time prompt is a per-computer preference.** Whether FreePair asks
  you to save a copy on close is remembered on the machine you set it on, not
  inside the tournament file.
