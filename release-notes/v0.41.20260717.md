# FreePair v0.41

This release is a focused **left-hand navigation tune-up**: the sections list is
tidier and quicker to work with, the Event Operations menu is out of the way
until you need it, and the player-search results size themselves sensibly.

## What's new in v0.41

- **Sections start collapsed.** Opening an event now shows a clean, scannable
  list of section names instead of every section expanded at once. Great for
  multi-section events where you just want to find the right section first.

- **Click a section to open it.** Selecting a section expands it so you can see
  its player count, rounds, and actions. Click the **same** section's name again
  to collapse it (and again to reopen) — no need to aim at the small chevron.

- **One click to expand or collapse everything.** Click the **Sections** heading
  to collapse all sections if any are open, or expand them all if they're closed.
  The up / down arrows next to the heading still work as explicit shortcuts.

- **Event Operations is now a menu.** *Renumber Boards*, *Pair All Sections*,
  *Sync All Rosters*, the *open-in-windows* views, *Publish*, and the *exports*
  now live in a single **Event Operations** dropdown that opens on demand — so it
  no longer pushes the sections list down when it's open.

- **Player search sizes the Name column to fit.** In the *Search from player DB*
  results, the **Name** column now auto-fits the actual names (no wasted empty
  space for short names) and you can still drag it wider if you like.

## Installer

- **Download:** [FreePair-win-x64-Setup.exe](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Setup.exe)
- **Portable zip:** [FreePair-win-x64-Portable.zip](https://github.com/freepairchess/FreePair/releases/latest/download/FreePair-win-x64-Portable.zip)
- **Auto-update:** Existing FreePair installs (v0.3 and later) will pick up
  v0.41 automatically. If you want the update now, click "Check for updates"
  in the Settings tab.

The installer bundles the .NET 10 runtime, the BBP Pairings FIDE engine,
and the rules-reference document — no additional downloads or configuration
required.

## Known limitations (still planned for later releases)

- **Section collapse state is per session.** How you've expanded or collapsed the
  sections list isn't saved with the event file yet — every open starts collapsed.
